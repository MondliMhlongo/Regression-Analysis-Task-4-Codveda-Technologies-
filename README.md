# Regression-Analysis-Task-4-Codveda-Technologies-

Regression Analysis (Task 4)
Overview

This project was completed as part of the CodeAlpha Data Analytics Internship Program. It demonstrates the application of Multiple Linear Regression to analyse relationships between variables and predict target values using real-world datasets.

Objectives
Build regression models using multiple datasets.
Train and evaluate predictive models.
Measure model performance using statistical evaluation metrics.
Compare actual and predicted values through visualisations.
Tools & Technologies
Python
Pandas
Scikit-learn
Matplotlib
Tasks Performed
Imported and preprocessed multiple datasets.
Selected independent and dependent variables.
Split datasets into training and testing sets.
Trained Multiple Linear Regression models.
Generated predictions on unseen data.
Evaluated model performance using:
R² Score
Mean Squared Error (MSE)
Mean Absolute Error (MAE)
Root Mean Squared Error (RMSE)
Visualised actual versus predicted values.
Learning Outcomes

Through this project, I gained practical experience in:

Regression modelling.
Predictive analytics.
Model evaluation and performance assessment.
Feature selection.
Interpreting regression results through statistical metrics and visualisations.
